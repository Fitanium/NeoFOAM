\doxysubsection{Detailed Description}
Represents a mesh for tree-\/based adaptive mesh refinement (AMR). 

The \mbox{\hyperlink{classNeoFOAM_1_1treeAMRMesh}{tree\+AMRMesh}} class provides functionality for refining, coarsening, balancing, and writing the mesh. It also provides information about the number of elements in the mesh.
