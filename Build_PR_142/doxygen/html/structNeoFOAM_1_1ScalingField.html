Detailed Description

template<typename ValueType>
struct NeoFOAM::ScalingField< ValueType >

A wrapper class that represents either a span of values or a single value.

This class is used to store either a span of values or a single value of type ValueType. It provides an indexing operator operator[] that returns the value at the specified index or a constant value and is used to scale fields in the DSL.

ValueTypeThe type of the values stored in the class.
