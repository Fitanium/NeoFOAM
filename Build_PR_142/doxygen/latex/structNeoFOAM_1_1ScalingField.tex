\subsubsection*{template$<$typename Value\+Type$>$\newline
struct Neo\+FOAM\+::\+Scaling\+Field$<$ Value\+Type $>$}

A wrapper class that represents either a span of values or a single value. 

This class is used to store either a span of values or a single value of type {\ttfamily Value\+Type}. It provides an indexing operator {\ttfamily operator\mbox{[}\mbox{]}} that returns the value at the specified index or a constant value and is used to scale fields in the \mbox{\hyperlink{namespaceNeoFOAM_1_1DSL}{DSL}}.
